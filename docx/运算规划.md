# 运算规划

## 大体思路

- 计算采用字符形式进行大数高精度运算
- 储存时进行算法压缩，保证尽可能小的储存空间

## 翻译器结构

```c++
// 整数和小数
struct Num
{
	// 已压缩的数字
	char* num;
	// num字节数
	size_t size;
	// 小数还是整数：0整数 1小数
	bool type;
	// 正负 0正 1负
	bool pm;
};
```
